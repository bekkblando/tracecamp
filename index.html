Overview

We focus on learning practical programming with hands-on web development. Specifically, we teach Python and Django, since they are flexible for creating a wide variety of applications.
Each free week-long boot camp teaches skills that are immediately useful for building web apps. Two sessions will be offered over Christmas break. All students will design and develop a web app to add to their portfolio. We welcome any major at any skill level.

What

An in person 7 day full time bootcamp focused on building Web Applications. Beginners are welcome as we will start with basic coding and move to Web Application Development over the 7 days.

When

There are two seperate sessions to pick from. Session 1 will be 12/17/18 - 12/23/18
Session 2 will be 12/31/18 - 1/6/19
Each day will start at 9 with lecture/live coding until 12, we will then break off into workshop time until each student finishes the daily assignment. Instructors will be available to help with assignments until 5.

Where

The workshops will be held on the Clemson campus.

Curriculum

Python, web technologies, and Django over the 7 days.

Day One

We will cover the basics of programming with Python. We will learn concepts that are transferable to any programming language.

Day Two

We will dive deeper into Python. We will learn the difference between object-oriented and functional programming.

Day Three

We will start learning about web technologies and how the Internet works behind the scenes. We will build a RESTful API.

Day Four

We will introduce a Python web framework named Django. We will also start learning the essentials of databases and model design.

Day Five

We will continue to practice Django basics and take an in-depth look at Django models. We will also cover HTMLâ€”the markup language for creating webpages.

Day Six

Each student will start designing and developing their personal projects. Optionally, we will learn how to style webpages with CSS and add interactivity to webpages with JavaScript.

Day Seven

We will focus on developing and finishing the student projects.
